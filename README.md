# Portfólio online

## Sobre o projeto
O projeto Portfólio online tem como objetivo mostrar o meu resumo profissional, o meu currículo, os meus projetos pessoais, conhecimentos e habilidades técnicas, certificados de cursos e contatos.

- Typescript
- Sass
- Bootstrap
- Docker
